# MOEADpy

This code uses 
* [scikit-learn](https://github.com/scikit-learn/scikit-learn)
* [pymop](https://github.com/msu-coinlab/pymop)

# Installation
Install with pip command.
```
pip install -U scikit-learn
pip install pymop
```

# Usage
Please set parameters and problem in config.yml 

Run MOEAD.py
```
python MOEAD.py
```
